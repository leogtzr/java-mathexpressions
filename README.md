MathExpressions It's a math library, recognizes and evaluate math expressions and returns the numeric value.

MathExpressions has 38 math functions like:

ABS, EXP, SIGN, SQRT, RAIZ, LOG, LN, LOG10, SIN, COS  
TAN, ASIN, ACOS, ATAN, FACTORIAL, COT, SEC, CSC, SINH
COSH, TANH, COTH, SECH, CSCH, ACSC, ASEC, ASINH, ACOSH  
ACSCH, ATANH, POWER, MAX, MIN, MOD, ACOT, ACOTH, LOG2  
RAND, ASECH.
